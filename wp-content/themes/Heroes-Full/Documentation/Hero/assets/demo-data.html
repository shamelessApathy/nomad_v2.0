Install Demo Data

NOTE! - Demo Data is by far the fastest way of getting going, though make sure you've first installed all required plugins by the theme. Next, remember that installing the demo data will not instantly give you a site as my demo, though you'll have that in about 5mins from reading this documentation! The benefit of demo data, is that you'll have a great understanding of the workings of the theme.

1. Install with FastWP WordPress Importer Plugin

Also be sure that you have to default plugins installed (Visual Composer and Contact Form 7 installed and activated) We have created a folder labeled "Demo Data" which contains an export for the Live Preview. The place to upload this XML file is in WordPress under Tools > Import.

Once you've arrived on the Import page, you will click on the "WordPress" link: Import posts, pages, comments, custom fields, categories, and tags from a FastWP WordPress export file..

From here you will just need click "Choose File" and select the demo-data.xml file. Then click "Upload file and import".

You will then be prompted to Assign Authors. You can read the description at the top as this explains everything. Note: Make sure you Import Attachments.

Check the box "Download and import file attachments". This will download attachments for images Note that the images from Live Preview demo. are replaced with placeholder images due copyright reason.

You now just need to wait for the importer to finish. This might take a few minutes depending on your hosting server speed.

2. Set Menus In Place

With All the Demo Data Installed, the last thing we need to do is set the menus.

To do this go to "appearance" => "menus", click the "manage locations" tab on the top and then set the locations as the screenshot below, click the save changes button.

Your website now looks just like the demo. Proced to the next section of the docs.
