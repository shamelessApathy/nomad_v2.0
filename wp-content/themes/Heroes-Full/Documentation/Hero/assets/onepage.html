One Page Explained

One Page websites are getting more popular and provide a fast way to develope and display your content. The entire page is composed from smaller elemnts called sections and each section is build from smaller elements / rows, columns, image, html code etc. Each section is in fact a page and you can add that page to menu, this way you can navigate betweens section directly from menu.

You should check our Youtube Channel for a live demo of section and menu items - it's much more easier than it seems.

Visual Composer

We choose the awesome plugin Visual Composer to help you create Hero content. This allow you to build, modify and edit the theme content without touching a single line of code. Also there are some custom blocks in Visual Composer designed specifically for Hero

You can easily recognize this blocks by their section - all Hero elements are in Theme Elements area as in screenshoot

It's possible also to use the standard Visual Composer blocks and elements but this have their default design and not a Hero specific design
