Creating portfolio Posts

Add a New "Portfolio Item;

Adding a new portfolio item is just a matter of clicking "Add New" under the portfolio custom post type. To insert your image you just need to Upload an image and then set that image as your Featured Image.

Make sure that each portfolio item has a title, some portfolio-categories assigned, a featured image, and some content.

There is no hardcoded content into a portfolio item, you have complete freedom to create your own content (except the ajax portfolio). We have included some prebuilt layouts that you can use by accesing the Visual Composer Template on a blank page (check our youtube videos)

In Order to display the portfolio into a section you should use the Visual Composer and add a Portfolio Section block into your Modules Row

You should check our Youtube Channel for a live demo
