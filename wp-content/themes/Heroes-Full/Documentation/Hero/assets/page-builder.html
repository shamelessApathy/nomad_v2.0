Page Builder Best Practices

The following is a collection of quick tips, and advice for using the included page builder. It's incredibly customisable, but you should definitely read the following:

  1. Try and save your work as frequently as possible in order to avoid loosing your work
  2. Phoenix is a minimal template try and keep it this way, overloading the theme with lots of elements will slow Live Composer
  3. A Note on moderation! Just because the page builder lets you choose different fonts for almost everything, doesn't mean it's a good idea. Use fonts, colours, borders etc. sparingly, the default styling is nice and minimal, so don't ruin it by adding every single option you get your hands on :)
  4. Most of all, have fun and experiment with Phoenix Theme and Live Composer
