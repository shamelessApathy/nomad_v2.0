Portfolio Shortcode Showcase

The Portfolio Shortcode controls the entire portfolio. The options for it are user friendly, and you have alot of options to choose from, for example:

Include - This option is used to include just the project ID's specified by the user.

Exclude - This option is used to exclude just the project ID's specified by the user.

Include Categories - Just like the include option for items, this one gives you the posibility to include just the categories specified by the user.

Overlay on Mobile - Activate/Deactive the overlay for the portfolio items on mobile devices such as smartphones or tablets.

Boxed Style - Choose the type of Portfolio layout, boxed or fullwidth.

Grid Style - Choose the style of the portfolio grid, from 2 columns to 4 columns and you can also choose to have an detailed layout for the portfolio.

Display Filters - Not much to say about this, it shows/hides the filters of your portfolio. Filters are practically categories in which the items are placed.

Label for all project - Type in the label for all of your projects, for example "Show all".

Extra Class - This option is used for styling, if you want to style a specific portfolio when you have multiple ones present on your website, you can use extra class, this will generate a specific class for that portfolio.

Animate Text - Animates the element text on scroll.

Animation Settings - By default it has a specified animation on it, you can change it with one of your own if you feel like it.

Animation Delay - The delay before the animation starts, not much else to say about this.
