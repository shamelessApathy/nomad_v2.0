Hero

Page Templates

Hero has a few page templates available:

  1. Default Template : this is a full width container and you can use this for full width section like counter, testimonial, portfolio etc.
  2. Boxed Template : this is a centred 1170px wide container that would be use for normal section and standalone pages.
  3. One Page Template - Template that should be used for One Page Setup this page should be empty and it is only a holder for you One page content
  4. Transparent Bg : Use this template for home section in order to get the radial overlay effect ofer images and this is a full width container.

Page Options

Each Page has a few options right above Page Attributes box: Parallax Letters and Section Spacing. we'll explain this options right now.

Parallax Letters

Use this metaboxes to creat up to 4 animate letters that will be placed on page background. For each letter you can select :

  • Stellar ratio - scrolling speed related to browser scrolling, for ex 2 will be twice the browser scrolling speed and 0.5 will be half (slower)
  • Offset - vertical offset
  • Margin left - in percent, this will alow to add space between letters

Section spacing

From here you can select a responsive padding for each section, this way you will have variable padding (larger on desktop and smaller on mobile devices)

  • Big-Padding : this will add a 200px top and bottom padding on desktop and 100px on mobile devices
  • Normal-Padding : this will add a 150px top and bottom padding on desktop and 100px on mobile devices
  • Small-Padding : this will add a 100px top and bottom padding on desktop and 50px on mobile devices

You can also overwritte the pading in case you need it using the Override pading inputs
