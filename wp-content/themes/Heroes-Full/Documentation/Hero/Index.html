Hero Presentation

Hero - Retina WordPress Template is a multipurpose template that can be used by a creative agency or as a personal portfolio for a freelancer.


It's a fully responsive one/multi paged template built on Bootstrap 3 Framework. It comes with many intro section versions - full screen background, full screen parallax slider, full screen video. It also has implemented a coming soon page (with countdown), to upload it until you get your job done, along with a 404 error page and an external single page for projects.
